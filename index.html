02. Pengantar Polimorfisme

Polimorfisme berasal dari kata poly yang berarti banyak dan morph yang berarti bentuk, sehingga dapat diartikan sebagai “banyak bentuk”.

Konsep polimorfisme adalah objek dari subclass dapat diperlakukan sebagai objek dari superclass. Hal ini memungkinkan satu tipe data digunakan untuk berbagai objek yang berbeda.

Sebagai contoh, Kucing, Anjing, dan Burung semuanya dapat dianggap sebagai Hewan.

Tipe Polimorfisme:

  • Compile-time polymorphism: Method overloading (tidak didukung dalam Dart)
  • Runtime polymorphism: Method overriding (didukung dalam Dart)

03. Polimorfisme dengan Inheritance

Polimorfisme dengan inheritance memungkinkan referensi dengan tipe parent digunakan untuk menampung objek dari class child.

Hal ini membuat satu tipe data dapat digunakan untuk berbagai objek yang memiliki hubungan pewarisan.

DART SOURCE CODE
// Nama   : Uliya Gumpari
// NIM   : 251410008

class Bentuk {
  void gambar() => print('Menggambar bentuk');
}

class Persegi extends Bentuk {
  @override
  void gambar() => print('Menggambar Persegi');
}

class PersegiPanjang extends Bentuk {
  @override
  void gambar() => print('Menggambar Persegi Panjang');
}

void main() {
  Bentuk b1 = Persegi();
  Bentuk b2 = PersegiPanjang();

  b1.gambar();
  b2.gambar();
}
}

04. Manfaat Polimorfisme

  • Fleksibilitas: Program dapat digunakan untuk berbagai jenis objek
  • Pengembangan mudah: Penambahan tipe baru dapat dilakukan tanpa mengubah banyak kode
  • Penggunaan ulang kode: Method yang sama dapat dimanfaatkan kembali dari class induk
  • Kemudahan perawatan: Perubahan pada program lebih mudah dikelola

06. Praktik 1 Polimorfisme dengan List

Sebuah list dengan tipe parent dapat menyimpan berbagai objek dari subclass yang berbeda.

Ketika dilakukan perulangan, setiap objek akan menjalankan method sesuai dengan implementasi yang dimilikinya masing-masing.

DART SOURCE CODE
// Nama   : Uliya Gumpari
// NIM   : 251410008
// Kelas : SI2A

class Hewan {
  void bersuara() {
    print('Hewan bersuara');
  }
}

class Kucing extends Hewan {
  @override
  void bersuara() => print('Meong');
}

class Anjing extends Hewan {
  @override
  void bersuara() => print('Guk guk');
}

class Burung extends Hewan {
  @override
  void bersuara() => print('Cuit cuit');
}

void main() {
  // List polimorfisme
  List daftarHewan = [
    Kucing(),
    Anjing(),
    Burung(),
  ];

  // loop & panggil method
  for (var h in daftarHewan) {
    h.bersuara();
  }
}
}

07.Pengantar Abstraksi

Abstraksi adalah konsep dalam Object Oriented Programming (OOP) yang digunakan untuk menyembunyikan detail implementasi dan hanya menampilkan bagian penting dari suatu objek kepada pengguna.

Dengan abstraksi, pengguna tidak perlu mengetahui bagaimana suatu proses bekerja, tetapi cukup mengetahui apa yang dapat dilakukan oleh objek tersebut.

Konsep Abstraksi:

  • Menyederhanakan sistem dengan menyembunyikan detail yang tidak diperlukan
  • Menampilkan hanya fitur atau fungsi utama
  • Menggunakan abstract class atau interface

Tujuan Abstraksi:

  • Membuat kode lebih sederhana dan mudah dipahami
  • Mengurangi kompleksitas program
  • Meningkatkan keamanan dengan menyembunyikan detail internal
  • Mempermudah pengembangan dan perawatan kode

Cara Menerapkan Abstraksi di Dart:

  • Menggunakan abstract class untuk membuat kerangka dasar
  • Mendefinisikan abstract method tanpa isi
  • Mengimplementasikan method tersebut di class turunan
  • Menggunakan implements untuk membuat interface

08. Abstract Class

Abstract class merupakan class yang berfungsi sebagai blueprint dan hanya dapat digunakan melalui class turunannya.

DART SOURCE CODE
// Nama   : Uliya Gumpari
// NIM   : 251410008
// Kelas : SI2A

abstract class Kendaraan {
  String? merk;

  void start();

  void stop() {
    print('Kendaraan berhenti');
  }
}

class Mobil extends Kendaraan {
  @override
  void start() {
    print('Mobil $merk menyala');
  }
}

void main() {
  Mobil m = Mobil();
  m.merk = "BMW"; // merk mobil
  m.start();
  m.stop();
}
}

10.Interface di Dart

Interface di Dart adalah cara agar sebuah class memaksa class lain untuk mengimplementasikan method tertentu menggunakan implements.

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

interface class Flyable {
  void terbang() => print('Terbang...');
}

class KupuKupu implements Flyable {
  @override
  void terbang() => print('Kupu-kupu terbang indah');
}

class Drone implements Flyable {
  @override
  void terbang() => print('Drone terbang cepat');
}

void main() {
  KupuKupu k = KupuKupu();
  Drone d = Drone();

  k.terbang();
  d.terbang();
}
}

11. Perbedaan Extends vs Implements di Dart

extends: Digunakan untuk mewarisi properti dan method dari class lain. Class turunan dapat menggunakan atau override method dari class induk.

implements: Digunakan untuk memaksa class lain mengimplementasikan semua method dari class interface. Method harus di-override, tidak mewarisi implementasi.

12. Static Members dan Method

Static digunakan untuk membuat variabel atau method yang bisa diakses tanpa membuat objek dari class. Biasanya dipanggil langsung dengan nama class.

// Nim    : 251410008

    return x + y;
  }
}

void main() {
  print('Nilai pi : ${MathUtils.pi}');
  print('Result : ${MathUtils.add(10, 20)}');
}
}

11. Praktik 2 Enkapsulasi Lengkap

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

class User {
  String _username;
  String _password;
  String _email;

  User(this._username, this._password, this._email);

  // Getter
  String get username => _username;
  String get password => _password;
  String get email => _email;

  // Setter dengan validasi
  set username(String value) {
    if (value.length >= 5) {
      _username = value;
    }
  }

  set password(String value) {
    if (value.length >= 8) {
      _password = value;
    }
  }

  set email(String value) {
    if (value.contains("@")) {
      _email = value;
    }
  }
}

void main() {
  var user = User("uliya123", "rahasia123", "uliya@gmail.com");

  // mencoba mengubah data
  user.username = "uliya123";
  user.password = "rahasia123";
  user.email = "uliya@gmail.com";

  // output
  print("User: ${user.username}, Pass: ${user.password}");
  print("Email: ${user.email}");
}
}

12. Encapsulation pada Login System

Encapsulation juga dapat digunakan dalam sistem login untuk melindungi data penting seperti password. Pada contoh ini, password tidak dapat diakses langsung dari luar class. Pengguna hanya dapat melakukan proses login atau mengganti password melalui method yang telah disediakan..

DART SOURCE CODE
// Nama   : Uliya Gumpari
// Nim    : 251410008
// Kelas  : SI2A

class User {
  String _username;
  String _password;

  User(this._username, this._password);

  String get username => _username;

  bool login(String username, String password) {
    return _username == username && _password == password;
  }

  void gantiPassword(String lama, String baru) {
    if (_password == lama && baru.length >= 8) {
      _password = baru;
      print('Password berhasil diubah');
    }
  }
}

void main() {
  final user1 = User('admin', 'password123');

  print(user1.login('admin', 'password123'));

  user1.gantiPassword('password123', 'passwordBaru123');
}
}

13. Latihan

  • Buat class RekeningBank dengan:
    • Field private: _nomorRekening, _saldo
    • Method: deposit(), tarik(), cekSaldo()
    • Validasi: saldo tidak boleh negatif
  • Buat program sederhana untuk simulasi transaksi.
DART SOURCE CODE
// Nama   : Uliya Gumpari
// NIM   : 251410008
// Kelas : SI2A

class RekeningBank {
  late String _nomorRekening;
  late double _saldo;

  RekeningBank(this._nomorRekening, this._saldo);

  void deposit(double jumlah) {
    if (jumlah > 0) {
      _saldo += jumlah;
      print('Deposit berhasil: Rp $jumlah');
    } else {
      print('Jumlah deposit tidak valid');
    }
  }

  void tarik(double jumlah) {
    if (jumlah > _saldo) {
      print('Saldo tidak mencukupi');
    } else if (jumlah <= 0) {
      print('Jumlah penarikan tidak valid');
    } else {
      _saldo -= jumlah;
      print('Penarikan berhasil: Rp $jumlah');
    }
  }

  void cekSaldo() {
    print('Saldo saat ini: Rp $_saldo');
  }
}

void main() {
  var rekening = RekeningBank("123456789", 1000000);

  print('--- Simulasi Transaksi ---');

  rekening.cekSaldo();
  rekening.deposit(500000);
  rekening.cekSaldo();
  rekening.tarik(300000);
  rekening.cekSaldo();
}
